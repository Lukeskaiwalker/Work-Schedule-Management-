# v1.0.0 - First Public Release

Date: 2026-02-23

## Highlights

- Admin update center with release check, dry-run, install action, and manual fallback commands.
- Major project/task/planning workflow improvements (archive-aware behavior, completed task archive, calendar planning extensions).
- Project classes/templates (CSV import/export) with class-based task/material bootstrap.
- Project finance and hours improvements (planned vs reported visibility).
- Materials module with state lifecycle (`Order`, `On its way`, `Available`).
- Construction report, mobile usability, and weather localization/caching improvements.
- Additional migrations, tests, and operational/security documentation updates.

## Upgrade Notes

- Upgrading to this release is intended to be non-destructive for existing DB/file data.
- Run migrations after code update: `docker compose run --rm api alembic upgrade head`.
- Rebuild stack after update: `docker compose up -d --build`.
- Never run `docker compose down -v` during standard upgrades.

## Validation Baseline

- `./scripts/test.sh` passes on this release branch.
