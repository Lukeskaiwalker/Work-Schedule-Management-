# v1.6.0 - Admin Center RBAC controls, push notifications, and workspace redesign

Date: 2026-03-12

## Highlights

- Expanded Admin Center capabilities:
  - added a dedicated `Admin Center` entry in the sidebar user menu,
  - introduced a roles tab with editable permission templates for non-admin roles,
  - added per-user permission overrides from the users tab,
  - blocked destructive admin-role edits and self-lockout paths.
- Improved admin UX and browser compatibility:
  - redesigned the Admin Center layout,
  - fixed notification-panel clipping against the sidebar,
  - corrected Safari/WebKit spacing issues in permission grids and group member checklists,
  - restored the role reset action icon and tightened control labeling/readability.
- Added browser notification support:
  - task/message push notifications in the web app,
  - PWA manifest and iOS-specific notification guidance,
  - fallback notification display path for iOS PWA environments.
- Refreshed main workspace views:
  - calendar and weekly planning redesigned as unified grid views,
  - overview/dashboard received a lighter visual rework.
- Extended materials image retrieval:
  - improved Unielektro direct article lookup,
  - hardened fallback behavior to skip broken `/brand/` CDN paths,
  - preserved faster search/image/cart UX improvements already introduced after `v1.5.0`.

## Upgrade Notes

- No new Alembic migration files are introduced in this release.
- Recommended local/network rollout:
  - `BACKUP_PASSPHRASE='<passphrase>' ./scripts/safe_update.sh --pull --branch main`
