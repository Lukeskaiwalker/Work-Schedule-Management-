# v1.7.0 - Sidebar refresh, time-access overhaul, NRW absences, and release hardening

Date: 2026-03-15

## Highlights

- Refreshed the main workspace shell:
  - introduced the new dark navy sidebar redesign,
  - kept the existing responsive workspace/navigation behavior from earlier releases.
- Expanded time-tracking capabilities and controls:
  - added permission-based time access handling,
  - added workspace lock support and absence-type metadata,
  - included NRW public-holiday coverage in time calculations,
  - restored CSV timesheet export compatibility and corrected XLSX Soll/Ist/Differenz/Pausenzeit output.
- Hardened project-scoped access after the permission refactor:
  - employees remain limited to their explicit project memberships for project/material/WebDAV access,
  - CEO role again has required-hours management access,
  - employee project finance read access remains available for project workflows.
- Stabilized material-catalog image sync:
  - preserved cached/local catalog image URLs in the expected sync phases,
  - kept direct Unielektro article-number lookup as a fallback path without regressing state reporting.
