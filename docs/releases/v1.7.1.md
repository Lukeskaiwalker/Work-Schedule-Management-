# v1.7.1 - Construction report UX fixes, XLSX corrections, and safer attachment recovery

Date: 2026-03-17

## Highlights

- Corrected time-export totals:
  - XLSX export totals and `Pausenzeit` now match the UI overview,
  - monthly work-hours total uses the same corrected calculation path.
- Tightened the construction report workflow:
  - Enter no longer submits the form unexpectedly,
  - Enter in material rows adds a row instead of submitting,
  - project selection uses a searchable combobox,
  - long PDF text now wraps correctly.
- Hardened encrypted file access:
  - preview/download now validates stored encrypted payloads before chunked streaming,
  - corrupted stored payloads return a clean `409` API error instead of aborting the upstream response mid-stream.

## Upgrade Notes

- No new Alembic migrations are introduced after `v1.7.0`.
- Recommended rollout:
  - `BACKUP_PASSPHRASE='<passphrase>' ./scripts/safe_update.sh --pull --branch main`
